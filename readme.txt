== Description ==

**Superwp Post Notify Users** automatically notifies users with specified roles whenever a post is published or updated. Easily select which user roles to notify, choose post types, and customize the notification emails, all within the plugin's user-friendly settings page.

== Features ==

- **User Role Notifications**: Notify specific WordPress user roles (e.g., Editor, Subscriber) on post publication or updates.
- **Post Type Selection**: Choose which post types (including custom post types) trigger notifications.
- **Customizable Email Notifications**: Personalize email subjects and content with placeholders for post titles, URLs, and more.
- **Scheduled Notifications**: Set a delay (in hours) for sending notifications after a post is published or edited.
- **Multi-Language Support**: Includes translations in the `languages` folder.
- **Simple Configuration**: All options are accessible from a user-friendly settings page in WordPress admin.
- **Actively Maintained**: Regular updates to ensure compatibility with the latest WordPress versions.

== Installation ==

1. Upload the `superwp-post-notify-users` directory to `/wp-content/plugins/`.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Go to **Settings** > **Superwp Post Notify Users** to configure the notification settings.

== Usage ==

1. Navigate to **Settings** > **Superwp Post Notify Users** after activating the plugin.
2. Choose the user roles to receive notifications.
3. Select the post types that should trigger notifications (e.g., posts, pages, or custom post types).
4. Customize the email subject and content, using placeholders such as `[post_title]` and `[post_url]`.
5. Optionally, set up a delay for sending notifications after a post is published or updated.
6. Save the settings, and your users will start receiving notifications as configured.

== Frequently Asked Questions (FAQs) ==

**Q: Can I notify multiple user roles?**  
A: Yes, you can select multiple user roles for notifications.

**Q: Can I customize the notification emails?**  
A: Yes, the plugin allows you to fully customize both the subject and content of the notification emails.

**Q: Can I schedule notifications to be sent later?**  
A: Yes, you can specify a delay (in hours) before notifications are sent after a post is published or updated.

**Q: Are custom post types supported?**  
A: Absolutely! You can notify users about any registered post type, including custom post types.
